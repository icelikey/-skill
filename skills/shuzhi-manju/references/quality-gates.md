# Quality Gates

## Before Submitting A Board

- 上一段真实尾帧已查看。
- 第一张参考图是上一段尾帧。
- 第二张参考图是当前故事板。
- 当前提示词的 0-3 秒已按真实尾帧重写。
- 旁白关键词都有对应画面动作。
- 最后一镜明确服务下一段首帧。
- 提示词正文没有本地路径、费用记录、平台参数。
- 音频参考只用于声线一致，不复述旧内容。

## Tail Frame Review

检查尾帧：

- 主体清晰，不糊、不崩、不遮挡。
- 光线与色彩可延续。
- 构图中有下一条开场可抓住的主体。
- 没有字幕、乱码、logo、水印、可读长文字。
- 没有偏离用户禁用元素。

## Prompt Rewrite Rules

当尾帧不完全匹配下一条故事板：

- 不否定尾帧，解释它如何过渡到下一条主题。
- 0-3 秒先稳定承接尾帧。
- 3-6 秒再逐步引入下一条故事板主体。
- 不要突然跳到新空间。

## Thread Compression Rules

长任务中：

- 不读取完整状态历史，除非排查状态错乱。
- 不反复加载长 Skill 文档。
- 巡检只记录短输出。
- 每完成一段，只保留本段状态、尾帧描述、下一段桥接修改。
