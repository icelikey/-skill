# Prompt Template

```text
旁白：
...

导演目的：
把本段旁白拆成可见动作推进；第一镜承接上一段真实尾帧，最后一镜服务下一段首帧。

视觉风格：
明媚写实真实世界 + 宏大冷调科幻 + 克制东方宇宙隐喻。不要 logo、可读文字、PPT 信息图或未授权人物肖像。

连续性：
入场：严格承接上一段真实尾帧：描述主体、光线、色彩、空间结构。
出场：最后 1 秒稳定锁定，可作为下一段第一镜。

镜头计划：
0-3秒：承接真实尾帧，不跳切、不重新构图。
3-6秒：揭示本段核心结构。
6-11秒：完成尺度/空间/概念跃迁。
11-15秒：减速锁定下一段可用尾帧。

摄影机与镜头：
写清焦段、机位高度、运动路径、焦点变化和视觉结果。禁止泛泛写“慢推、环绕、俯冲任选”。

灯光与色彩：
...

声音设计：
必须生成本段中文旁白和统一音效；参考音频只锁定音色、语速、混响、音效质感和响度，不复述参考音频文本。不要音乐、字幕、水印或画面文字。

负面约束：
不要字幕，不要可读文字，不要 UI 乱码，不要水印，不要 logo，不要人物肖像，不要卡通，不要 PPT 科普感，不要古风堆砌，不要神怪角色，不要画面崩坏。
```
